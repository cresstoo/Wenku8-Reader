# 轻小说文库阅读器

一个油猴脚本，优化轻小说文库（wenku8.net）阅读页面的体验，支持多种字体、字号、主题配色，去广告。

## 功能特性
- 极简UI，自动隐藏广告和杂项
- 支持字体、字号、主题配色切换
- 一键回到顶部，双击自动滚屏
- 简繁切换

## 安装方法
1. 安装 [Tampermonkey](https://www.tampermonkey.net/) 或 [Violentmonkey](https://violentmonkey.github.io/)
2. 打开 [脚本主页](https://greasyfork.org/scripts/542843)，点击安装
